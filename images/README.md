# Images Folder

## Contents

This folder contains visual assets for the retail SQL Case Study project.

### Files:

1. **er-diagram.png** - Entity-Relationship Diagram
   - Shows the database schema
   - Displays relationships between tables
   - Includes all 8 tables: customers, sellers, orders, order_items, products, payments, reviews, geolocation

## Adding the ER Diagram

**To add your ER diagram:**

1. Take the screenshot of your ER diagram from the uploaded image
2. Save it as `er-diagram.png`
3. Place it in this folder
4. The README.md will automatically display it

**Current diagram shows:**
- Table relationships via foreign keys
- Primary key connections
- Data flow between entities
- Color-coded tables for easy identification

## Alternative: Generate Diagram from Schema

If you need to regenerate the diagram, you can:

1. Use BigQuery schema export
2. Import into a tool like:
   - draw.io (https://draw.io)
   - dbdiagram.io (https://dbdiagram.io)
   - Lucidchart (https://lucidchart.com)
3. Export as PNG and save here

## Usage

The ER diagram is referenced in:
- README.md (main project documentation)
- SETUP.md (setup guide)
- Documentation files

Make sure the image is:
- Clear and readable
- High resolution (at least 1200px wide)
- PNG or JPG format
- Shows all table relationships
